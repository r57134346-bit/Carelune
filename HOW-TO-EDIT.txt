CareLune editing guide

Recommended setup for public website + editable admin:

1. Upload this whole folder to a GitHub repository.
2. Connect that GitHub repository to Netlify.
3. In Netlify, enable Identity.
4. In Netlify, enable Git Gateway.
5. Open your live site at /admin to edit products.
6. Product edits will update data/products.json in GitHub and Netlify will republish the public site.

Local editing before publishing:

Option A: Use the admin panel
1. Open admin.html.
2. Add, edit, or delete products.
3. Click "Export products.js".
4. Replace the old products.js in this folder with the downloaded products.js file.
5. Open index.html to preview.
6. Upload the whole CareLune folder again to update the public website.

Option B: Edit directly
1. Open products.js.
2. Replace the title, category, link, image, summary, bestFor, and date values.
3. Save the file.
4. Open index.html to preview.
5. Upload the whole CareLune folder again to update the public website.

Categories currently supported:
- Skincare
- Bodycare
- Haircare

Important:
The /admin folder is for Netlify + GitHub editing. The admin.html file is a local fallback editor.
